DuckDuckGo Search Model Context Protocol Server on Databricks Apps

Your DuckDuckGo search MCP server is up and running on Databricks Apps.

Get Started

Connect to the Server

Connection Details

Server URL
https://your.app.url.databricksapps.com/mcp/
Transport
Streamable HTTP
Example Connection (Python) 

from databricks.sdk import WorkspaceClient
from databricks_mcp import DatabricksOAuthClientProvider
from mcp.client.streamable_http import streamablehttp_client as connect
from mcp import ClientSession

client = WorkspaceClient()

async def main():
    # Connect to a streamable HTTP server

    app_url = "https://your.app.url.databricksapps.com/mcp/"
    async with connect(app_url, auth=DatabricksOAuthClientProvider(client)) as (
        read_stream,
        write_stream,
        _,
    ):
        # Create a session using the client streams
        async with ClientSession(read_stream, write_stream) as session:
            # Initialize the connection
            await session.initialize()
            # Call the search tool
            tool_result = await session.call_tool("search", {"query": "Python programming", "max_results": 5})
            print(tool_result)
